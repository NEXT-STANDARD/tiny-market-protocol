## なぜ存在するのか

2025年から2035年にかけて、3つの力が資本市場を再構築しています。

1. **ソロプレナーはAIツールにより従業員1人あたり売上を5.7倍に引き上げている**
   （AIネイティブスタートアップ平均 $3.48M/人 vs 従来SaaS $610K/人）
2. **伝統的VCはDPI危機で崩壊しつつある**
   （2018ビンテージのVCファンドは歴史標準0.8xに対し実績0.6x）
3. **IPO市場は1990年代の数分の一に縮小**
   （米国：1980-2019年中央値158件/年 → 2025年90件、上場時年齢中央値13-14年）

既存の資本スタックは、**200人規模から2,000人へ成長する企業**を前提に
設計されています。次の10年の勝者は、**少人数のまま巨額売上へ到達するチーム**
です。

彼らのための市場が必要です。

このリポジトリはその市場の形を描きます。

## TINY と AGORA の区別

このリポジトリは2つの補完的アイデアを探求します。

- **TINY** — 急進的な上場基準を持つ上場先（従業員10人以下、ARR $500K以上、
  黒字、VC調達額$10M未満）。小ささを美徳とする取引所。
- **AGORA** — 人間投資家が**意図（Intent）**を表明し、自律AIエージェントが
  24/7/365で執行する、エージェント主導取引層。

両者は独立して存在しうる。両者は組み合わされうる。本プロトコルは両者を
仕様化します。
