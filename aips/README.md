# AIPs — Improvement Proposals

This directory collects all proposals that refine, extend, or contradict the
TINY Market Protocol. The numbering follows the Bitcoin BIP / Ethereum EIP /
Nostr NIP convention.

## Reading Order

Start with AIP-001. Later AIPs assume familiarity with it.

## Status Definitions

- **Draft** — Under active discussion. May change.
- **Accepted** — Endorsed by current maintainers as a reference model.
- **Rejected** — Considered and not adopted. Kept for historical reference.
- **Superseded** — Replaced by a later AIP. Link to the successor.

## Types

- **Philosophy** — Argues a claim about why or what.
- **Protocol** — Specifies technical mechanics.
- **Jurisdiction** — Describes legal/regulatory pathways.
- **Research** — Presents data or analysis.
- **Meta** — About the AIP process itself.
